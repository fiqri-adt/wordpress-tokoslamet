Woocommerce Kode Pembayaran adalah Plugin khusus Woocommerce yang digunakan untuk memberikan kode unik dibelakang total transfer calon pelanggan agar Anda mudah mengelola proses pengecekan pembayaran.

== Description ==

Woocommerce Kode Pembayaran menambahkan sebuah kode unik berupa 3 digit angka dibelakang total belanja calon pelanggan Anda. Sehingga Anda dapat dengan mudah membedakan transfer masuk dr pembeli, bahkan puluhan transfer yg masuk secara bersamaan dapat di identifikasi oleh Anda.

Dengan beragam fitur yang ditawarkan, plugin ini sungguh powerful dan menjadikan plugin ini wajib digunakan ke semua toko online yang menggunakan wordpress.

== Installation ==

Dibawah ini adalah 3 cara mudah untuk melakukan instalasi pada plugin WooCommerce Kode Pembayaran

1. Untuk menginstal plugin caranya cukup mudah. Apabila Anda ingin mengupload langsung menggunakan file zip. Melalui wordpress pada menu Plugins > Add New > Upload Plugin
2. Aktifkan Plugin
3. Halaman pengaturan plugin di WooCommerce > Kode Pembayaran
4. Atur konfigurasi sesuai kebutuhan.
5. Have fun!

Kami menyediakan [dokumentasi](http://docs.agenwebsite.com/products/woocommerce-kode-pembayaran "Dokumentasi WooCommerce Kode Pembayaran") untuk Anda apabila mengalami kesulitan dalam menginstall atau mengupdate plugin WooCommerce Kode Pembayaran.
